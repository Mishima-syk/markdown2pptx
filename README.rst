=============
markdown2pptx
=============

`markdown2pptx` is PowerPoint slide generator from markdown txt

Requirements
------------
* Python 3.6 or later
* python-pptx
* click

Features
--------
* convert markdown txt to pptx

Setup
-----
clone this repo and python setup.py install
